# Creando un proyecto

Intalar paquetes Proyecto!

```bash

npm install

pnpm install

yarn install
```

## Desarrollo

Una vez que hayas creado un proyecto e instalado las dependencias con npm install (o pnpm install o yarn), inicia un servidor de desarrollo:

```bash
npm run dev

```
